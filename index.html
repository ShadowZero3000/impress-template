Presentation Template

A simple way to write impress.js presentations

Need a way to make a nice presentation without thinking too hard on the impress.js stuff? This is it!
By: Josh Souza

Who am I?

Josh Souza
  • HTML 5 enthusiast
  • Developer (So many languages)

Why?

  • Tired of PowerPoint
  • Needed a consistent theme for presentations
  • Liked Impress.js
  • Didn't like having to think through layouts

What

  • Framework/template for being able to just start writing content
  • Javascript to handle the layout automatically if you don't want to
  • Permits you to still put in your own layouting
  • Is far from perfect

Cool facts

  • You can load up other fun content using javascript, because it's all HTML
